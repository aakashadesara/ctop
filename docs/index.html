Features

Everything you need, nothing you don't.

Zero dependencies. Single file. Instant startup. Just the essentials for keeping tabs on your Claude sessions.

>_

Real-time monitoring

CPU, memory, status, and uptime for every Claude Code session. Auto-refreshes on a configurable interval.

[=]

Context window tracking

Visual bar showing input, cache, output, and free tokens out of the 200k context window — color-coded by saturation.

$

Cost estimation

Per-session and total cost calculated from token usage and model pricing. Supports Opus, Sonnet, and Haiku.

~

Sparkline graphs

Inline CPU and memory history sparklines rendered right in the process list. See trends at a glance.

/

Sort & filter

Sort by CPU, memory, context usage, or age. Filter by branch, model, directory, slug, or title.

x

Process control

Kill individual sessions or nuke them all. Graceful SIGTERM or force SIGKILL. Clean up zombies in one keystroke.

vim

Vim-style navigation

Navigate with h/j/k/l, jump with g/G. Two view modes: list and pane grid. Feels like home.

{}

Configurable settings

Tune refresh interval, context limit, and default view via ~/.ctoprc or CLI flags. Your terminal, your rules.

Keyboard-driven

No mouse required.

Every action is a single keystroke away. If you know vim, you already know ctop.

j/k Navigate up/down
s Cycle sort mode
/ Filter processes
x Kill selected
P Toggle pane view
? Show help
r Force refresh
q Quit

Get started

Install in seconds.

Requires Node.js 18+ and macOS or Linux.

Homebrew

$ brew tap aakashadesara/ctop
$ brew install ctop-claude

npm

$ npm install -g ctop-claude
$ ctop

Run without installing

$ npx ctop-claude

Or clone from source: git clone https://github.com/aakashadesara/ctop.git
